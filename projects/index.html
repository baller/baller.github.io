01 / Controllable Portrait Lighting Editing

Identity-preserving portrait lighting control

This project explores controllable portrait lighting editing for effects such as hair light, Tyndall light, side light, sunset light, and slit light. The research direction emphasizes identity preservation, background consistency, controllable lighting strength, and robust instruction-following for diffusion or flow matching based editing systems.

  • Design lighting edit tasks where the subject identity and background should remain stable.
  • Use strength control as an explicit interface, from S0 to S1.0, instead of a binary edit/no-edit setting.
  • Prepare evaluation views for prompt adherence, artifact control, identity similarity, and background preservation.

02 / Sparse-view 3D Gaussian Rendering

WaveSplat-style sparse-view novel view synthesis

This research direction focuses on sparse-view 3D Gaussian rendering for indoor and remote telepresence settings. The core idea is to move from limited camera observations to stable novel view rendering with depth priors and Gaussian field optimization.

  • Model sparse multi-camera observations as the starting point for scene reconstruction.
  • Use geometry and depth priors to reduce ambiguity in under-constrained indoor scenes.
  • Represent the final system as a rendering pipeline that can be evaluated with novel target viewpoints.

03 / SuperTravelAgent

Multi-agent travel planning system

SuperTravelAgent is a multi-agent travel planning system that decomposes complex travel needs into manageable planning tasks. Based on the public repository, it integrates a React + TypeScript interface, FastAPI backend, real-time WebSocket communication, MCP service management, and tool access for maps, train tickets, social content, web search, file handling, and PowerPaint-style image editing.

  • Six-agent workflow: task analysis, decomposition, planning, execution, observation, and summarization.
  • MCP service matrix covering Baidu Maps, 12306 train tickets, Xiaohongshu content, real-time search, filesystem tools, HTTP tools, and PowerPaint.
  • Modern web application architecture with Ant Design UI, session management, model configuration, and real-time reasoning visibility.
